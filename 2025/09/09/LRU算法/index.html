LRU算法

本题为力扣 146. LRU 缓存：
请你设计并实现一个满足 LRU (最近最少使用) 缓存 约束的数据结构。

题目要求

实现 LRUCache 类：

  • LRUCache(int capacity)：以正整数作为容量 capacity 初始化 LRU 缓存。
  • int get(int key)：如果关键字 key 存在于缓存中，则返回关键字的值，否则返回 -1。
  • void put(int key, int value)：如果关键字 key 已经存在，则变更其数据值 value；如果不存在，则向缓存中插入该组 key-value 。
    如果插入操作导致关键字数量超过 capacity ，则应该逐出最久未使用的关键字。

要求：函数 get 和 put 必须以 O(1) 的平均时间复杂度运行。

解题思路

分为三步：

  1. 构造函数
    初始化 LRUCache 类，维护好容量即可。

  2. 核心思想
    整个算法就是实现两个主要操作：get 和 put。

    • 只要有数据被访问（无论是读还是写），都属于“最近使用”，放到链表头。

  3. 具体操作

    • GET：查找 key → value，如果没找到返回 -1；如果找到了，移动该节点到链表头。
    • PUT：
      • 如果 key 存在，更新 value 并移动到链表头。
      • 如果 key 不存在，判断容量是否已满：
        • 若已满，删除链表尾部节点（最久未使用）。
        • 插入新的 (key,value) 到链表头。

代码实现 (C++)

#include <list>
#include <unordered_map>
using namespace std;

class LRUCache {
public:
    LRUCache(int capacity) {
        this->capacity = capacity;
    }
    
    int get(int key) {
        auto it = map.find(key);
        if (it == map.end()) {
            return -1;
        }
        // 移动到链表头
        cache.splice(cache.begin(), cache, it->second);
        return it->second->second;
    }
    
    void put(int key, int value) {
        auto it = map.find(key);
        if (it != map.end()) {
            // 更新 value，并移动到链表头
            it->second->second = value;
            cache.splice(cache.begin(), cache, it->second);
        } else {
            // 容量已满，删除尾部
            if (cache.size() == capacity) {
                auto del = cache.back();
                map.erase(del.first);
                cache.pop_back();
            }
            // 插入新节点到头部
            cache.emplace_front(key, value);
            map[key] = cache.begin();
        }
    }

private:
    int capacity;
    list<pair<int,int>> cache; // 双向链表：头=最近使用，尾=最久未使用
    unordered_map<int, list<pair<int,int>>::iterator> map; // 哈希表：key -> 链表节点
};
